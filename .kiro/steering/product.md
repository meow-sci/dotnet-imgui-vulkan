# Product Overview

This is a minimal .NET 10 console application demonstrating ImGui integration using Hexa.NET bindings. The application creates a simple GUI window with basic text rendering to showcase the integration of ImGui with Vulkan rendering and GLFW windowing.

## Key Features
- Minimal viable ImGui application
- Cross-platform GUI rendering
- Modern graphics API integration (Vulkan)
- Lightweight dependency footprint

## Target Use Case
Serves as a foundation or reference implementation for developers wanting to integrate ImGui into .NET applications with Vulkan rendering backend.
